# Changelog

All notable changes to TestFly SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [Unreleased]

### Added
- Initial release of TestFly SDK
- UI automation with Playwright
- API testing with Playwright APIRequestContext
- TestNG integration
- Cucumber BDD support
- Allure and ExtentReports integration
- Thread-safe parallel execution
- Scenario Context for data sharing
- JSON, CSV, Excel data readers
- Type-safe configuration with Owner API
- Comprehensive action interfaces

### Features
- Multi-browser support (Chrome, Firefox, WebKit)
- Headless and headful execution modes
- Auto-wait mechanisms for UI stability
- Auto-retry for API calls
- JSON Path querying with Jayway
- POJO mapping with Jackson
- Screenshot capture on failures
- Detailed test reporting
- Type-safe configuration management
